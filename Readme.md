Desafio : Crie um sistema que gerencie seminiarios

O sistema deverá cadastrar Seminarios , estudantes, professores e local onde será realizado

Um aluno poderá está em apenas um seminário
Um seminario poderá ter nenhum ou varios alunos
um professor poderá ter nenhum ou vários alunos
um professor poderá ter varios seminários
um seminario deve ter um local

Campos básico (excluindo relacionamentos)
seminários : título
aluno : nome e idade
professor : nome, especialidade
local : endereço
